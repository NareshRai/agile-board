# Agile-board

Its basic Agile board created using react and for styling used boostrap.
It include -

 - Agile dashboard
 - Can add new card dynamicly for respective board
 - Responsive design
 - cross browser support

 Steps to run project
 1. Download Repo
 2. Run npm install
 3. Run npm start 
 4. Browser it works on localhost:3000
